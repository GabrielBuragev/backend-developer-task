    Introduction

    Welcome to my API documentation.

    Authenticating requests

    This API is authenticated by sending an Authorization header in the form "Basic {credentials}". The value of {credentials} should be your username/id and your password, joined with a colon (:), and then base64-encoded.

    Request

    GET folder

    Store a newly created folder in storage for the authenticated user.

    requires authentication

    Example request:

    curl -X POST \
    "http://localhost/folder" \
    -H "Authorization: Basic {YOUR_AUTH_KEY}" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -d '{"name":"Folder 1"}'

    Example response (200):

    {
    "name": "Folder 1",
    "user_id": 2,
    "updated_at": "2021-05-17 12:58:28",
    "created_at": "2021-05-17 12:58:28",
    "id": 7
}

    Request

    POST folder/{folder_id}/note

    URL Parameters

    folder_id  string
    The id of the folder.

    Body Parameters

    name  string
    The name of the note.

    is_public  integer optional
    The visibility status of the note, default: false.

    noteable_type  string
    The value must be one of text or list.

    noteable  object
    Object which holds note content data

    If noteable_type is set to text following rules apply

    Body Parameters

    noteable.content  string
    Text content of the text note

    If noteable_type is set to list following rules apply

    Body Parameters

    noteable.items[]  array
    Array of items for the list note

    noteable.items[].content  string
    Text content of each list item

    Display the specified note.

    requires authentication

    Example request:

    curl -X GET \
    -G "http://localhost/folder/1/note/1" \
    -H "Authorization: Basic {YOUR_AUTH_KEY}" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json"

    Example response (200):

    {
    "id": 1,
    "name": "My note 1",
    "is_public": 0,
    "noteable_type": "text",
    "created_at": "2021-05-17 14:15:15",
    "updated_at": "2021-05-17 14:15:15",

        "id": 1,
